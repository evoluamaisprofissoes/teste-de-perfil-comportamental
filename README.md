# Teste de Perfil Comportamental (Liderança) • Evolua+ Profissões

## Como publicar no GitHub Pages
1. Envie **tudo desta pasta** para a raiz do seu repositório.
2. Garanta que a logo esteja em: `assets/logo-evolua.png`
3. Ative o GitHub Pages em *Settings → Pages* (branch `main` / `/root`).

## Observações
- O botão **Calcular Resultado** fica no final do teste.
- Ao calcular, o site rola automaticamente até a seção de resultados.
- Se faltar alguma pergunta, o sistema destaca as questões pendentes.
